# 🚀 Claude Code 부트캠프에 오신 것을 환영합니다!

## 여기서 시작하세요

Claude Code를 완전히 마스터하기 위한 여정을 시작하신 것을 축하합니다. 이 부트캠프는 여러분을 완전 초보자에서 전문가 수준의 사용자로 안내합니다.

---

## 📋 빠른 시작 체크리스트

### 1단계: 환경 확인
```bash
# Claude Code가 설치되어 있는지 확인
claude --version

# 설치되지 않은 경우, 설치하세요
npm install -g @anthropic-ai/claude-code
```

### 2단계: 학습 경로 선택

| 경로 | 대상 | 기간 |
|------|------|------|
| **빠른 시작** | 빠르게 시작하고 싶은 분 | 2주 |
| **전문가 과정** | 체계적으로 배우고 싶은 분 | 4주 |
| **마스터 과정** | 모든 것을 마스터하고 싶은 분 | 6-8주 |

### 3단계: 첫 번째 레슨 시작

👉 **다음 파일을 열어주세요**: `lessons/beginner/section-1-getting-started.md`

## 🎯 학습 목표

이 부트캠프를 완료하면 다음을 할 수 있습니다:

### 초급 수준
- ✅ Claude Code 설치 및 설정
- ✅ 기본 파일 도구 사용 (Read, Write, Edit, Glob, Grep)
- ✅ Git 통합 활용
- ✅ CLAUDE.md 파일 작성

### 중급 수준
- ✅ 서브에이전트 활용
- ✅ 커스텀 스킬 생성
- ✅ 훅(Hooks) 설정
- ✅ 병렬 작업 관리

### 고급 수준
- ✅ MCP 서버 연결 및 구축
- ✅ 커스텀 서브에이전트 생성
- ✅ IDE 통합 최적화
- ✅ 플랜 모드 활용

### 전문가 수준
- ✅ 플러그인 개발
- ✅ 기업 배포
- ✅ 성능 최적화
- ✅ 보안 및 컴플라이언스

## 📖 권장 학습 순서

### 1주차: 기초
```
1일차: 레슨 1-3 (시작하기)
2일차: 레슨 4-5 (권한 및 워크플로우)
3일차: 레슨 6-8 (파일 도구)
4일차: 레슨 9-12 (검색 도구)
5일차: 레슨 13-17 (Git 통합)
주말: 프로젝트 1 완료
```

### 2주차: 초급 완료
```
레슨 18-25 완료
연습문제 실습
CLAUDE.md 작성 연습
```

### 3-4주차: 중급
```
레슨 26-50 학습
스킬 및 훅 생성
중급 프로젝트 완료
```

### 5-8주차: 고급 및 전문가
```
레슨 51-100+ 학습
MCP 서버 구축
기업 배포 계획
```
