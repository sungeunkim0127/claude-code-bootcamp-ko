# 연습 1: Claude Code 첫걸음

**소요 시간**: 30분
**사전 조건**: Claude Code 설치 완료

## 설정

```bash
mkdir ~/claude-exercises/ex1
cd ~/claude-exercises/ex1
echo "console.log('Hello');" > app.js
claude
```

## 과제

### 과제 1: 기본 읽기 (5분)

Claude에게 요청: "app.js 읽고 무슨 일을 하는지 설명해줘"

**예상 결과**: Claude가 파일을 읽고 설명함

**확인**:
- [ ] 파일 내용이 표시됨
- [ ] 설명이 정확함

---

### 과제 2: 파일 생성 (5분)

Claude에게 요청: "utils.js 파일을 만들고 greet(name) 함수를 작성해줘"

**예상 결과**: utils.js가 생성됨

**확인**:
- [ ] 파일이 존재함
- [ ] greet 함수가 올바르게 작성됨

---

### 과제 3: 파일 편집 (10분)

Claude에게 요청: "greet() 함수에 에러 처리를 추가해줘"

**예상 결과**: 유효성 검사나 try-catch 추가됨

**확인**:
- [ ] Edit 도구가 사용됨
- [ ] 에러 처리가 추가됨

---

### 과제 4: 다중 작업 (5분)

Claude에게 요청: "두 파일을 모두 읽고 어떻게 함께 사용할 수 있는지 설명해줘"

**예상 결과**: 두 파일을 분석하고 통합 제안

**확인**:
- [ ] 두 파일 모두 읽힘
- [ ] 통합 방법 제안됨

---

### 과제 5: Git 기초 (5분)

Claude에게 요청: "Git 초기화하고 첫 커밋 만들어줘"

**예상 결과**: .git 생성, 커밋 완료

**확인**:
- [ ] .git 폴더 존재
- [ ] 커밋 생성됨

---
